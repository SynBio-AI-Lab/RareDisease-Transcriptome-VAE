# Model Checkpoints

This directory is reserved for trained Variational Autoencoder (VAE) checkpoints (`.pt`).

Model files are not included in this GitHub repository due to size limitations.

Please download the Zenodo archive and extract it at the root of this repository.  
The model checkpoints will be placed here automatically.

Zenodo DOI: https://doi.org/10.5281/zenodo.18431155
